## 📖 Описание программы

**Expense Tracker** — это графическое приложение для отслеживания личных расходов. Программа позволяет:

- ✅ Добавлять расходы с указанием суммы, категории и даты
- ✅ Просматривать все расходы в удобной таблице
- ✅ Фильтровать расходы по категории и периоду дат
- ✅ Подсчитывать общую сумму расходов с учётом фильтров
- ✅ Сохранять данные в JSON-файл и загружать их при запуске
- ✅ Удалять записи
- ✅ Валидировать ввод данных
